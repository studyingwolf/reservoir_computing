## Installation
1. Clone this repo
```bash
git clone https://github.com/studyingwolf/reservoir_computing
```

2. Create a Python Environment
3. Activate Enviroment
```bash
source .venv/bin/activate
```

4. Install requirements with uv
```bash
pip install uv
uv sync
```

Note: Requirements are 
```bash
ipykernel>=7.2.0
```
